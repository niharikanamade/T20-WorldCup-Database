# 🏏 T20 World Cup Database

## Overview

This project is a relational database built using MySQL for managing T20 World Cup cricket data.

### Features

- Team Management
- Player Information
- Match Details
- Innings
- Batting Scorecards
- Bowling Statistics
- Ball-by-Ball Data
- Playing XI
- Match Officials
- Match Conditions
- Achievements

### Technologies Used

- MySQL
- SQL

### Database Tables

- Team
- Player
- T20_Match
- Match_Team_Role
- Playing_XI
- Innings
- Batting_Scorecard
- Bowling_Details
- Ball_By_Ball
- Dismissal
- Substitution
- Umpire
- Match_Umpire
- Match_Conditions
- Achievement_Type
- Match_Achievement
